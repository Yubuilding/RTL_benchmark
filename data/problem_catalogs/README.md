# External Problem Catalogs

This folder records external benchmark sources that are relevant to the project.

Policy:

- Only mirror full problem content into `benchmarks/` when the upstream license is explicit and compatible.
- If the upstream site exposes problems publicly but does not publish an open-source or content license, keep a link-only index here instead of copying problem text.
- If an open-source repository contains data derived from another source with unclear redistribution terms, mark it for manual review before mirroring.

Current catalogs:

- `hdlbits_index.json`: link-only index for HDLBits pages and topic groups
- `open_rtl_benchmarks.json`: external open-source RTL benchmark repos with license and import notes
